# jetbot-traffic-sign-ai

This project involved working with a NVIDIA Jetson Nano robot. The goal was to train the robot so it could successfully complete a loop around a circuit with different traffic signs and respond correctly to each sign. The signs included:
- Stop sign: robot stops for three seconds, then continues
- Slow down sign: robot reduces speed and returns to original speed after passing a stop sign
- Green traffic light: robot continues moving
- Yellow traffic light: robot stops until the light turns green
- Sheep: robot stops until the sheep is out of frame

This project involved a lot of iterations of data collection, data preprocessing, model training and evaluation, and general code which could control the robot.
